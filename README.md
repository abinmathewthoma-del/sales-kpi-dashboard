# Sales KPI Dashboard

A professional Sales KPI Dashboard designed to visualize business performance metrics including revenue growth, customer acquisition, conversion rates, regional sales analytics, and top-performing products.

## Features
- Revenue Analytics
- KPI Monitoring
- Sales Performance Tracking
- Responsive UI
- Interactive Charts
- Regional Insights
- Customer Growth Metrics
- Modern Dashboard Design

## Technologies Used
- React.js
- Tailwind CSS
- Recharts
- Vite
- JavaScript
